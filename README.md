# ZenTask | Precision Task Management

ZenTask is a professional-grade task management system built with Django. It features a premium, modern dashboard designed for high-density productivity, moving away from generic to-do list aesthetics.

## 🚀 Professional Features

- **Strategic Dashboard**: Real-time analytics showing total, pending, and completed objectives.
- **Milestone Tracking**: Categorize tasks into distinct operational sectors.
- **Priority Architecture**: Low, Medium, High, and Urgent priority tiers with distinct visual coding.
- **Deadline Management**: Integrated due dates for every objective.
- **Premium UX**: Glassmorphic interfaces, modern typography (Outfit/Inter), and desktop-class sidebar navigation.

## 🛠️ Technology Stack

- **Framework**: Django 6.0
- **Styling**: Pure CSS (Custom Design System)
- **Icons**: Lucide-inspired SVG System
- **Database**: SQLite (Development)

## 📦 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sominenivenkatarajesh/To-Do-List.git
   ```

2. Set up a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   ```

3. Run migrations:
   ```bash
   python manage.py migrate
   ```

4. Start the server:
   ```bash
   python manage.py runserver
   ```
